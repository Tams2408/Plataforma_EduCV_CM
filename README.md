# AgriCV — Sistema de Recomendação Agrícola
Sistema que recomenda culturas agrícolas para Cabo Verde.
Autores: Natalino Semedo, Tamiris Évora e David Santos

## Como instalar
1. Instalar Python 3 (https://python.org)
2. Instalar SWI-Prolog (https://swi-prolog.org)
3. Instalar a biblioteca: pip install pyswip

## Como executar
Versão terminal:    python main.py
Versão com janela:  python main_gui.py

## Ficheiros
conhecimento.pl - base de dados Prolog
main.py         - versão terminal
main_gui.py     - versão com interface
